# 학돌함 v0.1.4

학교를 잘 돌아가게 하는 행정 자동화 도구를 한곳에 모아두는 포털입니다.

## 이번 버전 반영

- `제출모아`를 문서 자동화 도구로 추가
- 제출모아 링크는 `https://github.com/sen-vip/jechul-moa`로 연결
- 학돌T 브리핑 영역을 캡처 톤에 맞춰 크림 배경, 노란 테두리, 갈색 텍스트 계열로 정리
- 메인 히어로 높이와 카드 간격을 줄여 첫 화면 피로감 완화
- 도구 카드 그리드를 `auto-fit`으로 조정해 화면 폭에 따라 자연스럽게 정렬
- GitHub처럼 앱 실행 링크가 아닌 경우 `actionLabel`로 버튼 문구를 바꿀 수 있게 개선

## 디자인 개선대책

1. **학돌T는 안내자 역할만 한다**  
   학돌T 브리핑은 화면 상단에서 짧게 알려주는 역할로 유지하고, 캐릭터를 크게 키우지 않습니다.

2. **색은 캡처 톤 유지**  
   배경 `#fff8e8`, 테두리 `#f4d58d`, 텍스트 `#5f4714` 계열을 사용해 따뜻한 노란 톤을 유지합니다.

3. **첫 화면은 덜 무겁게**  
   기존 보석상자 히어로는 유지하되 높이를 줄여 도구 목록으로 빨리 내려갈 수 있게 했습니다.

4. **도구 카드는 정보 위계를 고정**  
   아이콘 → 도구명 → 설명 → 태그 → 상태/버튼 순서를 유지해 새 도구가 늘어도 흐트러지지 않게 했습니다.

## 구성

- `index.html` : 메인 페이지
- `style.css` : 디자인
- `app.js` : 도구 목록, 검색, 필터, 즐겨찾기 기능

## 도구 공개/비공개 수정 방법

`app.js`의 `tools` 배열에서 `url`과 `status`만 바꾸면 됩니다.

### 공개 도구

```js
status: "운영중",
url: "https://sen-vip.github.io/example/"
```

### GitHub 저장소 연결 도구

```js
status: "공개저장소",
url: "https://github.com/sen-vip/example",
actionLabel: "GitHub"
```

### 비공개 도구

```js
status: "비공개",
url: ""
```

### 블로그로 안내할 도구

```js
status: "블로그 예정",
url: ""
```

블로그 글이 생기면 `url`에 블로그 주소를 넣으면 됩니다.
