# 학돌함 v0.1.11

학교를 잘 돌아가게 하는 행정 자동화 도구를 한곳에 모아두는 포털입니다.

## 이번 버전 반영

### 학돌T 안내 위치 정리
- 상단 학돌T 안내바 제거
- 학돌T 안내를 `오늘의 책상` 안쪽으로 이동
- 문구: `🐥 할 일과 메모는 이 브라우저에만 저장돼요.`
- 화면 위쪽을 덜 밀어내도록 작은 안내칩 형태로 정리

### 빈 상태 안내 정리
- 바로 꺼내기 빈 상태 문구에 작은 🐥 적용
- 검색 결과 없음 영역을 더 작고 조용하게 조정
- 모바일에서도 학돌T 안내가 화면을 과하게 차지하지 않도록 조정

## 의도

학돌T는 큰 캐릭터가 아니라  
필요한 순간에만 조용히 알려주는 작은 안내자로 사용합니다.
