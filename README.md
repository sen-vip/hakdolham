# 학돌함 v0.1.14

학교를 잘 돌아가게 하는 행정 자동화 도구를 한곳에 모아두는 포털입니다.

## 이번 버전 반영

### 파비콘 적용
- 업로드본에 포함된 `favicon.svg`를 실제 파비콘으로 연결
- 기존 `favicon.png` 연결을 제거하고 SVG 기준으로 정리

```html
<link rel="icon" href="./favicon.svg" type="image/svg+xml" />
```

### 자주 쓰는 도구 자동 추천
- `열기` 버튼을 누른 횟수를 브라우저에 저장
- 2번 이상 연 도구는 `자주 꺼낸` 탭에 자동 표시
- 마지막으로 연 도구는 `최근 사용` 탭에 표시
- 별표 고정 도구는 `별표 고정` 탭에 별도로 표시
- 저장 위치: 브라우저 `localStorage`
- 서버 전송 없음

### 화면 순서 조정
1. 히어로 + 검색
2. 바로 꺼내기
3. 추천 도구
4. 내 책상
5. 전체 도구
