# 학돌함 v0.1.1

학교를 잘 돌아가게 하는 행정 자동화 도구를 한곳에 모아두는 포털입니다.

## 이번 버전 반영

- 초근누구 링크 추가
- 안 샀다 링크 추가
- 품샷은 공개 링크 제거
- 세콤매니저는 블로그 예정 상태로 표시
- 링크가 없는 도구는 `비공개`, `블로그 예정`, `준비중`으로 표시

## 구성

- `index.html` : 메인 페이지
- `style.css` : 디자인
- `app.js` : 도구 목록, 검색, 필터, 즐겨찾기 기능

## 도구 공개/비공개 수정 방법

`app.js`의 `tools` 배열에서 `url`과 `status`만 바꾸면 됩니다.

### 공개 도구

```js
status: "운영중",
url: "https://sen-vip.github.io/example/"
```

### 비공개 도구

```js
status: "비공개",
url: ""
```

### 블로그로 안내할 도구

```js
status: "블로그 예정",
url: ""
```

블로그 글이 생기면 `url`에 블로그 주소를 넣으면 됩니다.
