# 학돌함 v0.1.13

학교를 잘 돌아가게 하는 행정 자동화 도구를 한곳에 모아두는 포털입니다.

## 이번 버전 반영

### 공개형 첫인상 정리
- 히어로 문구를 공용 도구 포털 느낌으로 정리
- 처음 온 사람도 바로 볼 수 있는 `추천 도구` 영역 추가
- 추천 도구: 우리학교 생활달력, 공문핏, 계약 콕검, 학교카드 사용처 지도

### 내 책상 위치 정리
- `오늘의 책상` → `내 책상`
- 내 책상을 `바로 꺼내기` 아래로 이동
- 금액 한글 변환을 내 책상 첫 번째 카드로 배치
- 할 일/메모는 개인 기능처럼 조용하게 정리

### 금액 한글 변환 예시 변경
- 예시 금액을 `1,234,560`으로 변경

## 화면 흐름

1. 히어로 + 검색
2. 추천 도구
3. 바로 꺼내기
4. 내 책상
5. 전체 도구
