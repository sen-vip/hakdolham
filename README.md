# 학돌함 v0.1.19

학교 행정 자동화 도구를 한곳에 모아두는 포털입니다.

## 이번 버전

- 카테고리 탭과 대형 카드 목록 제거
- 업무분류별 `표형 행 + 3열 중형 카드` 구조 적용
- 우리학교 키박스를 최상단 핵심 도구로 유지
- 검색 결과를 검색창 바로 아래에 즉시 표시
- 검색 결과 최대 6개 표시 및 Enter로 첫 번째 도구 실행
- 검색어와 일치하는 문구 강조
- `계약 콕검`을 계약·지출·카드 분류로 이동
- 초근누구·세콤매니저 변환기를 점검·확인 분류에 통합
- 자주 쓰는 도구 중복 영역 제거
- 업무별 도구를 나의 책상보다 위로 이동
- 데스크톱 3열, 태블릿 2열, 모바일 1열 반응형 적용
- 할 일·메모·금액 변환·뽀모도로 기능 유지

## 배포

`index.html`, `style.css`, `app.js`, `favicon.svg`를 저장소 루트에 업로드하면 GitHub Pages에서 동작합니다.
