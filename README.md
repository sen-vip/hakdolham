# 학돌함 v0.1.5

학교를 잘 돌아가게 하는 행정 자동화 도구를 한곳에 모아두는 포털입니다.

## 이번 버전 반영

### 오늘의 책상 추가
- 오늘 할 일 추가
- 할 일 체크/삭제
- 완료한 할 일 삭제
- 빠른 메모 자동 저장
- 모든 내용은 브라우저 `localStorage`에만 저장

### 디자인 미세조정
- 즐겨찾기 영역 이름을 `앞칸에 꺼내둔 도구`로 변경
- 카드 높이와 여백을 조금 더 컴팩트하게 조정
- 카테고리 칩 여백과 톤 정리
- 검색창 placeholder를 업무명 중심으로 수정
- 계약 콕검 아이콘과 설명 톤 정리
- 히어로 영역을 조금 더 압축

## 사용 의도

학돌함을 단순한 도구 링크 모음이 아니라  
아침에 열어보는 작은 업무 책상처럼 쓰는 방향입니다.
