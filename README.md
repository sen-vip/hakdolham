# 학돌함 v0.1.18

학교 행정 자동화 도구를 한곳에 모아두는 포털입니다.

## 이번 버전

- 상단 히어로를 압축하고 검색을 중심 기능으로 강화
- 우리학교 키박스를 최상단 핵심 도구 및 전체 도구 첫 번째로 배치
- 키박스 링크를 `school-keybox` 주소로 정리
- 나의 책상을 2×2 동일 크기 구성으로 변경
- 25분 집중 / 5분 휴식 뽀모도로 타이머 추가
- 타이머가 다른 탭에서도 정확히 동작하도록 종료 시각 기반으로 계산
- 할 일·메모·금액 변환의 브라우저 저장 기능 유지
- 상황별 추천과 빠른 실행을 `자주 쓰는 도구`로 단순화
- 전체 도구 카드 높이와 여백 축소
- 프로토타입 전국 학사일정 캘린더 항목 제거

## 배포

`index.html`, `style.css`, `app.js`, `favicon.svg`를 저장소 루트에 업로드하면 GitHub Pages에서 동작합니다.
